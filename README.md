### Intern Project

## 💻 프로젝트 소개

인턴을 위한 과제용 프로젝트입니다.

### 개발 환경 및 도구

- Vite (React)
- Typescript
- Styled-CSS
- Tanstack-Query
- React-router-dom
- Zustand

## ⚙️ 개발 기간

2024.08.07(수) ~ 2024.08.11(토) | 실제 개발 기간 : 4일

## ⚒️ 주요 기능

- 회원가입 기능: id, 비밀번호, 닉네임을 입력하여 회원가입 할 수 있습니다.
- 로그인 기능: id, 비밀번호를 입력하여 로그인 할 수 있습니다.
- 마이페이지: 마이페이지로 이동하여 개인의 정보를 수정할 수 있습니다.
